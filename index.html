    Give it a name, why dont'cha?

    Project maintained by fnichol Hosted on GitHub Pages — Theme by mattgraham

    Nameit helps you generate a random name for a project, database, session, you decide. You get glorious - style names such as "pushy-clock" and "lovely-ducks" or ask for a random number on the end to give you names like "numberless-cactus-123" and "superb-shoes-915".

    Installation

    Add this line to your application's Gemfile:

    gem 'namit'

    And then execute:

    $ bundle

    Or install it yourself as:

    $ gem install namit

    Usage

    nameit Command

    Nameit ships with the nameit command so you can get a shiny new name right from the command line:

    $ nameit
abhorrent-flowers

    Ruby Library

    require 'nameit'

Nameit.generate
# => "sassy-clocks"
Nameit.generate
# => "dazzling-comb"

3.times.map { Nameit.generate }
# => ["ethereal-music", "historical-argument", "recondite-pear"]
